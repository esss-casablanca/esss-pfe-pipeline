# Manuscript Formatting Protocol

For each target journal, reformat V5 to its **current** instructions for authors without changing
its science. Work through this checklist, recording the journal's requirement and the action taken.

## Elements to align
- **Title page** — title (and any length limit), authors and affiliations, corresponding-author
  details, word counts, running head if required.
- **Abstract** — type (structured vs unstructured), required subheadings, word limit.
- **Keywords** — number and any MeSH requirement.
- **Section structure** — the journal's expected headings for a review (e.g. Introduction, Methods,
  Results/Synthesis, Discussion, Conclusion) and any mandated subsections.
- **Limits** — total word count, reference count, number of tables/figures.
- **Reference style** — convert to the journal's style (e.g. Vancouver, APA, ICMJE). Converting
  *style* is allowed; the set of sources and their identifiers must stay identical to the lock.
- **Tables & figures** — placement, captions, numbering, file/format rules.
- **Required declarations** — ethics/IRB statement, consent (if applicable), funding, conflicts of
  interest, data-availability, author contributions, acknowledgements.
- **Reporting-guideline checklist** — attach the relevant one (e.g. PRISMA 2020 / PRISMA-ScR) if the
  journal expects it.

## Lock-safe reformatting rules
- Never drop or alter a frozen claim, number, statistic, PRISMA count, citation, or reference.
- If the word/reference limit cannot be met without removing locked science, **stop and flag it** to
  the student; suggest a journal whose scope accommodates the work rather than cutting content.
- Where a required declaration depends on information only the student has (funding, ethics approval
  number), insert the appropriate `[... REQUIRED]` marker and list it in the submission checklist.

## Copyright
Do not paste the journal's author-guideline text into outputs. Read it, paraphrase the requirements,
and apply them.
