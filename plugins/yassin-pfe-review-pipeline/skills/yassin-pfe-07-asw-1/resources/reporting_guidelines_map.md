# Reporting Guidelines Map

Use this file to align revision with the appropriate health-research reporting family. Do not claim formal compliance unless the manuscript actually contains the required items.

| Manuscript type | Guideline family to consider | Revision focus |
|---|---|---|
| Observational cross-sectional/cohort/case-control study | STROBE | design, setting, participants, variables, bias, sample size, statistical methods, results, limitations |
| Randomized controlled trial | CONSORT | randomization, allocation, blinding, intervention, outcomes, flow, harms |
| Systematic review | PRISMA | eligibility criteria, databases, search strategy, selection, extraction, risk of bias, synthesis |
| Scoping review | PRISMA-ScR | objectives, concept/context/population, search, selection, charting, evidence map |
| Qualitative study | COREQ / SRQR | research team, reflexivity, sampling, setting, data collection, analysis, themes, credibility |
| Case report | CARE | patient information, clinical findings, timeline, diagnostic assessment, intervention, outcomes, patient perspective |
| Diagnostic accuracy study | STARD | participants, index test, reference standard, accuracy estimates, flow |
| Prediction model study | TRIPOD | predictors, outcome, sample size, missing data, model development/validation, performance |
| Quality-improvement report | SQUIRE | problem, available knowledge, rationale, intervention, measures, context, results, interpretation |
| Study protocol | SPIRIT when trial; PRISMA-P when review; appropriate protocol logic otherwise | objectives, design, setting, eligibility, data collection, analysis, ethics, dissemination |
| Clinical practice guideline | AGREE / RIGHT | scope, stakeholders, evidence base, recommendations, implementation, transparency |
| Survey study | CROSS / STROBE logic | sampling frame, recruitment, instrument development, validity, administration, response rate, analysis |

## Required marker

Use `[REPORTING ITEM REQUIRED]` when a reporting item is expected but missing.

## Caution

Reporting guidelines improve completeness. They do not justify inventing content. If information is missing, mark it instead of filling it.
